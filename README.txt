Instructions on running code

Before running code, hw02_dataset.mat must be in the same folder as the script.

Type in `python ds4400_hossain_script.py`

This should be python 3 so if default is python 2, then use python3 instead of python

A matplotlib plot with two subplots will show up

The first plot is the training data plotted with the decision boundary line marked and colors distinguishing the two classes 

The second plot is the testing data plotted with the decision boundary line marked and colors distinguishing the two classes 

The console output will be in this format showing the

1. Classification accuracy on training set
2. Classification accuracy on test set
3. Classification error on test set

in that order
